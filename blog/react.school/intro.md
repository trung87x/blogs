# ✅ **Thứ tự học React hợp lý nhất**

### **1️⃣ Intro to React**

→ Đây là nền tảng bắt buộc: Component, JSX, Props, State, Event, Effect.

---

### **2️⃣ User Interface (UI cơ bản)**

→ Phần này giúp bạn hiểu cách tổ chức UI trong React, cách đặt component, layout, cấu trúc UI.

---

### **3️⃣ Todo List Project**

→ Đây là project nhỏ để luyện:

- state
- event
- list & key
- controlled input

→ Rất tốt để "cứng tay" trước khi học library hoặc router.

---

### **4️⃣ React Router**

→ Học routing _sau khi_ đã biết UI và state, vì Router là phần quan trọng để làm app nhiều trang.

---

### **5️⃣ Redux Toolkit**

→ Học quản lý state nâng cao khi:

- app nhiều trang → Router đã biết
- UI/State cơ bản đã vững  
  → Đây là thời điểm vào Redux Toolkit là đẹp nhất.

---

### **6️⃣ Material UI (MUI)**

→ UI library dùng sau khi đã hiểu React & state management.  
→ Dùng để skin app cho đẹp.

---

### **7️⃣ Project – Material UI Dashboard**

→ Dùng MUI + React + Router + Redux Toolkit  
→ Đây là project lớn, bạn làm sau khi biết tất cả phần trên.
