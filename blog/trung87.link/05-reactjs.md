Chúng ta tiến tới cấp độ "đỉnh cao" trong lộ trình Front-end hiện đại. Khi dự án trở nên quá lớn, việc dùng JavaScript thuần để quản lý hàng nghìn dòng code sẽ trở nên cực kỳ khó khăn. Đó là lý do **React.js** ra đời.

Ở cấp độ này, độ khó không nằm ở cú pháp, mà nằm ở **Tư duy hệ thống (System Thinking)**.

## CẤP ĐỘ 5: REACT.JS (LIBRARY FOR WEB APPS)

### 1. ĐẠI CƯƠNG REACT.JS (The Component Philosophy)

_Thay đổi hoàn toàn cách bạn nhìn nhận một giao diện web._

- **Tư duy Component (Thành phần):** Thay vì viết một file HTML dài, bạn chia nhỏ giao diện thành các mảnh độc lập (như Lego) như `Button`, `Navbar`, `Card`. Bạn có thể tái sử dụng chúng ở bất cứ đâu.
- **JSX (JavaScript XML):** Một cú pháp đặc biệt cho phép bạn viết HTML trực tiếp ngay bên trong JavaScript. Điều này giúp logic và giao diện luôn đi đôi với nhau.
- **Virtual DOM:** React không cập nhật toàn bộ trang web khi có thay đổi, nó chỉ cập nhật đúng "mảnh ghép" bị tác động. Điều này giúp ứng dụng chạy cực nhanh.
- **Props & State:**
- **Props:** Cách truyền dữ liệu từ cha xuống con.
- **State:** "Trí nhớ" của Component, giúp nó biết khi nào cần thay đổi giao diện (ví dụ: khi người dùng mở một menu).

### 2. ỨNG DỤNG REACT.JS TRONG FRONT-END (Modern Application Building)

_Biến bạn từ một người làm web thành một kỹ sư xây dựng ứng dụng._

- **Ứng dụng 1: Quản lý trạng thái với Hooks (`useState`, `useEffect`)**
- Sử dụng Hooks để xử lý các hành động phức tạp như: Tự động tải dữ liệu khi vừa mở trang, hoặc ghi nhớ thông tin người dùng nhập vào mà không cần tải lại trang.
- _Kết quả:_ Ứng dụng hoạt động mượt mà như một phần mềm trên máy tính.

- **Ứng dụng 2: React Router (Điều hướng trang)**
- Cho phép người dùng chuyển từ trang "Trang chủ" sang "Sản phẩm" ngay lập tức mà trình duyệt không phải quay vòng tròn tải lại.
- _Kết quả:_ Tạo ra trải nghiệm **Single Page Application (SPA)** chuyên nghiệp.

- **Ứng dụng 3: Kết hợp hệ sinh thái (React + Tailwind)**
- Sử dụng Tailwind để Style cho các React Component. Đây là bộ đôi mạnh mẽ nhất hiện nay để xây dựng giao diện nhanh và đẹp.
- _Kết quả:_ Xây dựng được những hệ thống lớn như Dashboard quản lý, mạng xã hội thu nhỏ hoặc sàn thương mại điện tử.

### TỔNG KẾT MỤC TIÊU (Front-end Target)

Khi bạn làm chủ được **React.js**, kết quả hội tụ là:

> **Tầm vóc kỹ sư:** Bạn đã sở hữu bộ công cụ mạnh nhất hiện nay để đi làm tại các tập đoàn công nghệ. Bạn không chỉ viết code, bạn đang **xây dựng hệ thống**. Mọi kiến thức từ HTML, CSS, Tailwind đến JS đều hội tụ lại bên trong các React Component.
