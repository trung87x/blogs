# Ứng dụng 3: Tối ưu hóa SEO & Accessibility (Optimization)

```html
<!DOCTYPE html>
<html lang="vi">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Tiêu đề tối ưu SEO | Tên Thương Hiệu</title>
    <meta name="description" content="Mô tả nội dung trang web giúp tăng thứ hạng trên Google.">
    <meta name="keywords" content="frontend, html, seo, accessibility">
    <meta name="author" content="Tài Liệu Hệ Thống">

    <meta property="og:title" content="Tiêu đề hiển thị mạng xã hội">
    <meta property="og:description" content="Mô tả ngắn gọn khi share link.">
    <meta property="og:image" content="link-anh-thumbnail.jpg">
</head>
<body>

    <header>
        <h1>Tiêu đề chính (H1 - Quan trọng nhất cho SEO)</h1>
    </header>

    <main>
        <section>
            <h2>Tiêu đề phụ (H2)</h2>
            
            <img src="san-pham.jpg" alt="Mô tả chi tiết hình ảnh cho người khiếm thị và bot Google">

            <button aria-label="Đóng menu">X</button>

            <label for="search">Tìm kiếm:</label>
            <input type="search" id="search" name="q" placeholder="Nhập nội dung...">
        </section>
    </main>

    <footer>
        <p>© 2025 Thương hiệu của bạn. Bảo lưu mọi quyền.</p>
    </footer>

</body>
</html>

```
