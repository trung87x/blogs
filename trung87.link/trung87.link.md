## Css

### 1. Layout & Display: Xác định cách phần tử xuất hiện và căn giữa icon

.icon-box {
display: flex;
justify-content: center;
align-items: center;
position: relative;
}

### 2. Box Model: Kích thước và hình dạng

.icon-box {
width: 40px;
height: 40px;
margin: 10px;
padding: 5px;
}

### 3. Aesthetics: Màu sắc, Border và Background

.icon-box {
background-color: #f0f0f0;
border: 1px solid #ddd;
border-radius: 50%;
}

### 4. Interaction: Hiệu ứng khi di chuột hoặc click

.icon-box {
transition: all 0.3s ease;
cursor: pointer;
}

.icon-box:hover {
background-color: #007bff;
transform: scale(1.1);
}
